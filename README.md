#### JDBC의 문제점
    1. SQL을 실행하는 비용이 크다.
    2. SQL이 데이터베이스 마다 다르다.(유지보스 및 추가개발 대한 문제)
    3. 스키마를 바꾸면 코드가 바뀌는 경우가 많다.
    4. try 리소스와 같은 반복적인 코드가 상당량 늘어난다.
    5. 필요하지 않은 데이터도 모두 읽어 오는 경우가 있기 때문에 비효율 적이다.
